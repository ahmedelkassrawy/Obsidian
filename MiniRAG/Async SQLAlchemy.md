# Overview

You’re using **SQLAlchemy (async)** — specifically:

```python
from sqlalchemy.ext.asyncio import AsyncSession, async_sessionmaker
from sqlalchemy import select
```

So, **all async DB actions** use `AsyncSession` and `select()` from **SQLAlchemy**.
They allow **non-blocking queries, inserts, commits, and refreshes** inside an async FastAPI app.

---
## ⚙️ Libraries used for DB operations

| Library                                                    | Purpose                                                          |
| ---------------------------------------------------------- | ---------------------------------------------------------------- |
| **SQLAlchemy** (`sqlalchemy` and `sqlalchemy.ext.asyncio`) | ORM and async database layer                                     |
| **Databases** (optional, not used here)                    | Async database wrapper (alternative)                             |
| **Pydantic** (`pydantic`)                                  | Validation models for request/response (not DB operations)       |
| **FastAPI** (`fastapi`)                                    | Framework integrating async dependency injection (via `Depends`) |

# 🧠 Async DB Operations Extracted

Below is a clean summary of **every async DB operation** found in your code — where it’s used, what it does, and from what library it comes.

---

### 1️⃣ **Creating an Async Session**

```python
AsyncSessionLocal = async_sessionmaker(
    async_engine,
    class_=AsyncSession,
    expire_on_commit=False
)
```

* **Library:** `sqlalchemy.ext.asyncio`
* **Action:** Creates an **async session factory**.
* **Purpose:** Allows `async with AsyncSessionLocal()` to create a connection to your DB asynchronously.

---

### 2️⃣ **Dependency Injection for DB**

```python
async def get_db():
    async with AsyncSessionLocal() as session:
        yield session
```

* **Library:** `fastapi` + `sqlalchemy.ext.asyncio`
* **Action:** Creates and yields an **AsyncSession** per request.
* **Purpose:** Ensures each request gets its own DB session without blocking other requests.

---

### 3️⃣ **Executing Queries (SELECT)**

```python
result = await db.execute(select(Order).filter(Order.order_id == order_data["order_id"]))
```

* **Library:** `sqlalchemy`
* **Action:** Runs an asynchronous SQL SELECT statement.
* **Purpose:** Fetch data from the database without blocking the event loop.

✅ **General pattern:**

```python
result = await db.execute(select(Model).filter(Model.field == value))
```

---

### 4️⃣ **Fetching One Result**

```python
existing = result.scalar_one_or_none()
```

* **Library:** `sqlalchemy`
* **Action:** Retrieves **one ORM-mapped result** or `None`.
* **Purpose:** Cleanly handle the case where a record may or may not exist.

✅ **Alternatives:**

* `.scalars().all()` → get a list
* `.scalar_one()` → expect exactly one (throws error if not)

---

### 5️⃣ **Adding a New Record**

```python
db.add(db_complaint)
```

* **Library:** `sqlalchemy`
* **Action:** Adds an ORM object to the session (staged for insertion).
* **Purpose:** Marks an object as new; it will be inserted when `commit()` is called.

✅ **General pattern:**

```python
new_record = Model(**data)
db.add(new_record)
```

---

### 6️⃣ **Committing Changes**

```python
await db.commit()
```

* **Library:** `sqlalchemy.ext.asyncio`
* **Action:** Writes all pending changes to the database.
* **Purpose:** Finalizes INSERTs, UPDATEs, or DELETEs asynchronously.

✅ **Important:**
Always `await db.commit()` after modifying the DB, otherwise changes are not persisted.

---

### 7️⃣ **Refreshing an Object**

```python
await db.refresh(db_complaint)
```

* **Library:** `sqlalchemy.ext.asyncio`
* **Action:** Reloads the object from the database.
* **Purpose:** Ensures the local object has the latest state (like autogenerated IDs).

✅ **General pattern:**

```python
await db.refresh(obj)
```

---

### 8️⃣ **Reading Data (GET operations)**

```python
result = await db.execute(select(Complaint).filter(Complaint.id == complaint_id))
complaint = result.scalar_one_or_none()
```

* **Library:** `sqlalchemy.ext.asyncio`
* **Action:** Fetch a single row asynchronously.
* **Purpose:** Used for lightweight data retrieval.

✅ **You can also use:**

```python
results = (await db.execute(select(Complaint))).scalars().all()
```

---

### 9️⃣ **Updating Data**

```python
complaint.escalation_status = "Escalated"
await db.commit()
```

* **Library:** `sqlalchemy.ext.asyncio`
* **Action:** Updates a field and commits changes.
* **Purpose:** Modify existing records asynchronously.

✅ **Pattern:**

```python
object.field = new_value
await db.commit()
```

---

### 🔟 **Startup Data Initialization**

```python
@app.on_event("startup")
async def startup_event():
    async with AsyncSessionLocal() as db:
        await init_sample_data(db)
```

* **Library:** `fastapi` + `sqlalchemy.ext.asyncio`
* **Action:** Runs database logic asynchronously at app startup.
* **Purpose:** Pre-populates database if empty.
